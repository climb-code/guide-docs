# Guide Docs

Welcome to **Guide Docs**, a comprehensive, multi-language documentation platform built to provide high-quality learning resources for developers of all levels.

## 🚀 Overview

Guide Docs is a centralized hub for mastering modern web and software development. Whether you're learning your first programming language or diving into advanced DevOps patterns, we've got you covered.

## 🛠️ Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Documentation Engine:** [Starlight](https://starlight.astro.build/)
- **Styling:** Custom CSS & Vanilla UI
- **Deployment:** Vercel / GitHub Pages

## 📚 Topics Covered

Our curriculum is constantly expanding and currently includes:

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), TypeScript
- **Backend:** Node.js, Python, Java
- **Mobile:** Swift & SwiftUI
- **Databases:** MySQL, PostgreSQL
- **DevOps & Tools:** Git, Docker, AWS Deployment
- **Interview Prep:** Curated coding questions and DSA patterns

## 🏁 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18.14.1 or higher) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/climb-code/guide-docs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd guide-docs
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:
```bash
npm run dev
```
The site will be available at `http://localhost:4321`.
